## Quick Copilot Check

Before you begin, verify Copilot is working:

1. Look at the bottom-right of VS Code -- the Copilot icon should say "Ready"
2. Press `Ctrl+Shift+I` (or `Cmd+Shift+I` on Mac) to open Chat
3. Ask: "Hello, are you working?"
